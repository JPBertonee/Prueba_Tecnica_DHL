## Descripción del proyecto

Este proyecto corresponde a una prueba técnica de análisis de datos sobre un dataset de transporte.

El objetivo fue analizar la información disponible para obtener una visión general del perfil de envíos, identificando volúmenes, carriers, tiempos de entrega, lanes principales, tendencias semanales y distribución por peso facturable.

El análisis fue realizado utilizando SQL Server para las consultas y Power BI Desktop para la visualización de resultados.

---

## Herramientas utilizadas

- SQL Server
- Power BI Desktop
- Excel / CSV
- GitHub
